# 🏅 Global Olympic Medals Analysis Dashboard (1990–2012)

This project presents a comprehensive analysis of Olympic medal trends from 1990 to 2012 using Power BI. The dashboard provides insights into country-wise performance, athlete achievements, gender participation, and overall medal distribution across multiple Olympic events.

📊 Project Objectives

* Analyze global Olympic medal distribution from 1990–2012.
* Identify top-performing countries and athletes.
* Compare medal counts across different Olympic years.
* Examine gender-wise participation and achievements.
* Discover trends and patterns in Olympic performance over time.

🔍 Key Analysis Performed

* Country-wise Gold, Silver, Bronze, and Total Medal Analysis.
* Top Countries by Overall Medal Count.
* Year-wise Olympic Performance Trends.
* Athlete Performance Analysis.
* Gender-wise Medal Distribution.
* Sport-wise Medal Contribution.
* Ranking of Nations based on Medal Achievements.
* Interactive filtering and drill-down analysis for deeper insights.

🛠️ Tools & Technologies Used

* Power BI – Dashboard Development & Data Visualization
* Power Query – Data Cleaning & Transformation
* DAX – Measures, KPIs, and Calculated Columns
* Excel/CSV Dataset – Olympic Historical Data

📈 Dashboard Features

✔️ Interactive Slicers and Filters
✔️ KPI Cards for Medal Statistics
✔️ Dynamic Country Rankings
✔️ Trend Analysis Visualizations
✔️ User-Friendly and Professional Layout
✔️ Multi-Page Analytical Dashboard

🎯 Key Insights

* Identified the most dominant Olympic nations during 1990–2012.
* Analyzed medal-winning patterns across different years.
* Revealed gender participation trends in Olympic events.
* Highlighted sports contributing the highest number of medals.
* Compared country performances through interactive visual analytics.

🚀 Outcome

Developed an end-to-end Olympic analytics solution that transforms raw historical data into meaningful insights, enabling users to explore global Olympic performance trends efficiently through interactive visualizations.
